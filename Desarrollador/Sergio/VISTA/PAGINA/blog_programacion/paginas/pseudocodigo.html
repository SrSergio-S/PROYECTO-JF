Pseudocode

¿What is it?

The pseudocode is a way of writing the steps that a program is going to carry out in the closest way to the programming language that we are going to use later. It is like a false language, but in our language, in human language and in Spanish.

¿How Pseudocode Works?

The main objective of pseudocode is to represent the solution to an algorithm in the most detailed way possible, and in turn as similar as possible to the language that will later be used to encode it.

Pseudocode Structure

It begins with the keyword Process (or alternatively Algorithm are synonyms) followed by the name of the program. It is followed by a sequence (Sequential Control Structure) of instructions

Core items

Las palabras reservadas comúnmente utilizadas en pseudocódigo son:
Inicio, Fin.
Leer, Escribir.
Si, Entonces, Sino, FinSi.
Según, FinSegun.
Mientras, Hacer, FinMientras
Repetir, HastaQue.
Para, FinPara.

Characteristics

as principales características de este lenguaje son: ... Facilita el paso del programa al lenguaje de programación. Es independiente del lenguaje de programación que se vaya a utilizar. Es un método que facilita la programación y solución al algoritmo del programa.

Problems

To solve problems with pseudocode, the following aspects must be taken into account:
¿Does the user have to enter data?
¿Is there some kind of repetition inside the algorithm?
¿Is there any conditional structure?
¿What condition would be the most appropriate, if we have a loop or an If structure?
¿What variables do I need in the algorithm?
¿How should I initialize the variables?
¿What result should I print on the screen? (if necessary)
