Software Programming

Mathematical operators

Mathematical operators allow you to perform arithmetic operations on their operands. Each operation performed with the operands generates a result
that will be stored in some memory location. The type of the generated value, and therefore its internal representation, will depend on the types of the operands.

The hierarchy of operators

The hierarchy of the operators is the order in which mathematical operations are solved.

Relational operators

Relational operators are symbols used to compare two values. If the result of the comparison is correct, the considered expression is true, otherwise it is false.

Variables

A variable is a named data item whose value can change during the course of a program's execution. The variable name must follow the naming convention of an identifier (alphabetic character or number and the underscore).
When more than one variable is defined in a single declaration, the name must be separated by commas. Each variable declaration must end with a semicolon. Variable names cannot match a reserved word.

Decision cycles

A loop or cycle, in programming, is a sequence of code instructions that is executed repeatedly, until the condition assigned to said loop is no longer met.

Repetitive cycles

Repetitive control structures are those that allow a set of instructions to be executed several times, according to the value generated by the relational and/or logical expression.
